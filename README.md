Currency Formatter
=========

A small library that adds currency prefix to amounts

## Installation

  `npm install @ituple-public/currency-formatter`

## Usage

    var curFormatter = require('@jdaudier/currency-formatter');

    var formattedNum = curFormatter.rupee(35666.00);
  
  
  Output should be `Rs 35666.00`


## Tests
